# hAcKeR
speaks like a  ~~hacker~~ lamer

> ### how to use the tool:
> ```sh
> py main.py "simple text"
> ```
> output:
>```sh
> SiMpLe tExT
> ```

> ### how to add "1!1!!1!!":
> ```sh
> py main.py "simple text!"
> ```
> output:
> ```sh
> SiMpLe tExT!11!!1!!
> ```
> the exclamation point is converted to random numbers and exclamation points
